MAZUWO WIRELESS STATIC SITE

This is a Cloudflare Pages friendly static site.

AI helper:
- Lucy runs locally from assets/js/ai.js and assets/data/mazuwo-ai-model.json.
- It does not call /api/ai and does not need any API key.

Deploy:
- Build command: none
- Output directory: /

Forms:
- Application and contact forms open the visitor's email app with a prepared message.
- Coverage requests open WhatsApp with a prepared request.
